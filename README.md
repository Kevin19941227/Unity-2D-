# Quit

一款使用 Unity 開發的2D俯視角動作遊戲。

## 專案簡介
本專案是一款使用 Unity 製作的俯視角動作遊戲，核心內容包含角色移動、攻擊系統、怪物 AI、碰撞判定、UI 設計與場景切換等功能。  
此外，專案中也使用了 **Cinemachine** 進行鏡頭調整，讓遊戲畫面在操作與戰鬥時能有更穩定的視角表現。

## 專案特色
- 俯視角角色移動
- 玩家攻擊系統
- 怪物 AI 與戰鬥互動
- 玩家與怪物碰撞判定
- UI 系統
- 場景切換
- 使用 Cinemachine 進行鏡頭調整

## 我的負責內容
在此專案中，我主要負責遊戲程式設計與系統整合，包含：

- 玩家與怪物碰撞箱設定
- 玩家攻擊邏輯
- 怪物攻擊邏輯
- 基礎怪物 AI 實作
- 使用 Tilemap 建置 2D 地圖
- UI 設計與整合
- 場景切換設定
- 遊戲 Bug 修正與流程調整
- 使用 Cinemachine 進行鏡頭調整

## 使用技術
- Unity
- C#
- Cinemachine
- Tilemap

## 專案結構
- `Assets/`：遊戲資源、腳本、場景、Prefab、UI 資源
- `Packages/`：Unity 套件設定
- `ProjectSettings/`：Unity 專案設定

## 執行方式
1. 將此專案 Clone 下來
2. 使用對應版本的 Unity 開啟
3. 開啟主場景
4. 在 Unity Editor 中執行
